자동화는 규모보다 빈도가 더 중요합니다.

하루에 3번 반복하는 작업을 1주일 동안 한다면, 이미 자동화 후보입니다.
복잡한 시스템부터 만드는 대신 아래 질문으로 시작하면 보통 더 잘 맞습니다.

## 체크할 질문

1. 이 작업은 언제 가장 귀찮은가?
2. 사람이 직접 판단해야 하는 부분은 어디까지인가?
3. 실패했을 때 수동으로 복구하기 쉬운가?

## 개인 블로그에도 적용할 수 있는 자동화

- 글 템플릿 자동 생성
- 배포 시 sitemap 생성
- 이미지 최적화
- RSS 업데이트
- 최근 글 자동 정렬
